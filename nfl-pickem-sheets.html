üéØ How Scoring Works

Once games are completed:

  1. Update the Winner column in Games sheet (use "away" or "home")
  2. Script will automatically calculate wins/losses
  3. 1 point per correct pick
  4. +1 bonus point if all 3 picks are correct (4 total)

Note: You'll need to set up Google Apps Script to enable automatic scoring and write functionality. See the README for instructions.
